# Diário de Desenvolvimento — Etapa 3

## Objetivo

Implementar o gerenciamento de incidentes.

## Atividades

* Estruturação do armazenamento local dos incidentes.
* Implementação da página de histórico.
* Criação da tela de upload de logs.
* Desenvolvimento dos filtros e pesquisa.

## Exemplos de Prompts

* Criar um serviço para armazenamento de incidentes.
* Implementar pesquisa por título.
* Filtrar incidentes por severidade.
* Criar uma tabela de histórico.

## Resultado

A aplicação passou a permitir visualizar e organizar incidentes simulados.

## Dificuldades

Foi necessário revisar manualmente parte do código para manter consistência entre os componentes.
